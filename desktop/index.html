Work Log

日報を資産化する

その日の作業を記録しつつ、残したい情報だけを Markdown に正規化します。

保存先

日報は daily/YYYY/YYYY-MM-DD.md に保存されます。編集中の状態は .work-log-state に保持します。

使い方

  1. today に今日やることを追加
  2. 終わったらチェックを入れて impact を書く
  3. 保存するとチェック済みだけが done として出力

today

チェックを入れたものだけ Markdown の done に反映します。

support

1行に1項目で書いてください。

improvements

改善アイデアや次に試したいこと。

learning

学んだこと。任意です。

notes

メモや調査ログ。

Markdown プレビュー

保存内容のプレビューです。
